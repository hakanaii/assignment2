# 콘솔 To-Do List

간단한 콘솔 기반 할 일 관리 프로그램입니다. 할 일을 추가, 조회, 수정, 삭제하고 완료 여부를 관리할 수 있습니다. 모든 데이터는 tasks.json 파일에 저장되어 프로그램을 다시 실행해도 유지됩니다.


### 주요 기능

* 할 일 추가
* 할 일 목록 보기
* 완료/미완료 상태 전환
* 할 일 내용 수정
* 할 일 삭제
* 완료율 통계 보기
* JSON 파일로 자동 저장/불러오기

### 앞으로 추가하고 싶은 기능
* 우선순위(중요도) 설정

* 키워드로 할 일 검색

* 색상 출력
